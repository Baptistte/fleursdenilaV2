# Fleurs de Nila — Élégance Intemporelle

Site vitrine minimaliste et sophistiqué pour "Fleurs de Nila", fleuriste artisanal à Nîmes.

## 🌸 Concept

Design **minimaliste scandinave** inspiré des codes du luxe français et de l'élégance intemporelle. Une approche épurée qui met en valeur les créations florales artisanales à travers un site vitrine premium.

## 🎨 Direction Artistique

### Esthétique
- **Style** : Minimalisme scandinave avec raffinement français
- **Approche** : Épurée, sophistiquée, premium
- **Inspiration** : Sites haute couture et boutiques artisanales d'exception
- **Ton** : Élégant sans ostentation, luxueux mais accessible

### Palette de Couleurs
```
Bleu Profond    : #1a365d (primary)
Bleu Moyen      : #2d5282 (secondary)  
Bleu Clair      : #bee3f8 (accent)
Blanc Cassé     : #fafafa (cream)
```

### Typographie
- **Display** : Fleur de Leah (titres principaux, poétique)
- **Corps** : Georgia (élégant, lisible, intemporel)
- **Hiérarchie** : 
  - H1 : 5-8xl, font-light
  - H2 : 4-6xl, font-light
  - Body : lg-xl, font-light
  - Labels : sm, uppercase, tracking-widest

## ✨ Caractéristiques Design

### Layout
- **Espacement généreux** : Padding vertical important (py-24, py-32)
- **Grille fluide** : 2 colonnes sur desktop, 1 sur mobile
- **Max-width** : 7xl (1280px) pour confort de lecture
- **Marges** : px-6 mobile, px-12 desktop
- **Ratio images** : 4:5 pour collections, carré pour galerie

### Navigation
- **Fixed transparent** devenant opaque au scroll
- **Typographie fine** : uppercase, tracking-wide
- **Underline hover** : Animation subtile de soulignement
- **Menu mobile** : Fullscreen avec grandes typographies display

### Interactions
- **Transitions douces** : 300-700ms cubic-bezier
- **Hover subtils** : scale(1.05), translateY(-8px)
- **Animations scroll** : Intersection Observer pour fade-in
- **Lightbox galerie** : Overlay élégant avec fermeture fluide

### Boutons
- **Style minimaliste** : Bordure fine, padding généreux
- **Hover fill** : Animation de remplissage du background
- **Typography** : Uppercase, tracking-widest, font-light

## 🛠️ Technologies

- **HTML5** sémantique et accessible
- **Tailwind CSS** via CDN (configuration custom)
- **CSS3** pour animations et effets avancés
- **JavaScript Vanilla** optimisé pour performance
